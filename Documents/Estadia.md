# Adding a new Coloborator
1. Launch the Estadia Google Sheet;
2. Select the Addicionar Colaborador on the Estadia Menu
3. The syste displays a modal dialog box, with the following features:
   1. An X at the top right; it instructs the system to dismiss the modal dialog box;
   2. A Search, where the actor types a search string;
   3. A Selection Box, where the system shows the filtered names of the Pessoas;
   4. A Cancel button at the botton right; it instructs the system to dismiss the modal dialog box;
   5. An OK button at the the botton right; it instructs the system to pick the selected person and dismiss the modal dialog box;
4. The actor Selects the Pessoa:
   1. Types a search string the Search box, as the names in the Selection box are filtered, until he finds the name of the Pessoa he wants to add as a Collaborator
   2. Selects the name of the Pessoa he wants to add as a Collaborator
   3. Presss OK
5. The system inserts the Pessoa at the top of the Estadia!Dados sheet;
6. The Administrator finishes to setting the new collaborator's attributes at the the Estadia!Dados sheet;
